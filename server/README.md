The chuzr server is a node.js webservice providing HTTP endpoints
for accessing the chuzr data store.  Endpoints include:

 * GET /products?title=&categories=&skip=&limit=

 * GET /products/{id}

 * GET /user/{id}

 * POST /votes

 * POST /users

 * PUT /users/{id}

 * DELETE /users/{id}
 
 * PATCH /products/{id}
